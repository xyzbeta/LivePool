# Changelog

## v1.0.0 (2026-06-17)

### Features

- **采集 (Collect)**: 多源并发爬取（GitHub IPTV、公开 m3u8），本地种子文件导入，URL 自动去重
- **校验 (Validate)**: 异步高并发 HTTP 探测，四阶段深度检测（状态码 → 内容验证 → 媒体分段可达性 → NAL 视频流扫描），死链指数退避机制
- **过滤 (Filter)**: 智能去重加权评分系统（稳定性 ×50、视频完整性 ×8、分辨率 ×~4、延迟 ×~100）
- **分类 (Classify)**: 自动归类（央视、卫视、地方、海外等），支持 JSON 映射文件手动覆盖
- **生成 (Generate)**: 标准 IPTV m3u8 输出，支持按分组导出独立文件，Logo 本地缓存 + 防盗链
- **EPG 节目指南**: 外部 XMLTV 源代理，m3u8 头部自动附加 `url-tvg`，Token 保护访问
- **Web 管理面板**:
  - 仪表盘（统计概览、趋势对比、订阅地址）
  - 频道列表（搜索/筛选/排序/收藏/质量评分）
  - 采集源管理（CRUD、全量检测、进度可视化）
  - 用户管理（CRUD、2FA、邀请码、订阅拉取统计）
  - 调度配置（Cron 表达式、快捷预设、运行状态）
- **认证与安全**:
  - JWT 认证（HttpOnly Cookie + Bearer Header）
  - TOTP 二次验证（强制绑定、恢复码）
  - 邀请码注册系统（单次使用、使用者追踪）
  - 订阅 Token 访问控制
- **定时调度**: APScheduler 集成，支持 Cron 表达式，Web UI 可视化配置
- **Docker 部署**: Web + 调度器双容器架构，健康检查

### Bug Fixes

- 修复死链退避永不恢复的问题（skip 结果不递增 fail_count）
- 修复备份端点返回空用户的问题
- 修复 CORS 过滤导致 m3u8 输出孤立 EXTINF 行的问题
- 修复取消收藏后频道列表不刷新的问题
- 修复仪表盘进度步骤缺失 cache_logos 的问题
- 修复频道名/URL 的 XSS 漏洞
- 修复 `pre_auth` token 可当 session token 使用的问题
- 修复 `validate_invite_code` 导入缺失导致注册页实时校验失效的问题

### Optimizations

- 优化频道去重评分算法（延迟权重提升，CORS 加分移除）
- 优化 m3u8 写入为原子操作（临时文件 + rename）
- 优化管道步骤 UI（卡片式进度条、脉冲动画）
- 优化频道列表排版（URL 截断、质量评分展示）
- 优化用户注册页 UI（实时邀请码校验）
- 完善用户名服务端校验（字母数字中文白名单）
